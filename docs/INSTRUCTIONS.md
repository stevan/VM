```
ANY  = could be any one of the values below

NUM  = integer literal
STR  = pointer to static table
PTR  = pointer to heap
NULL = null pointer
BOOL = one of the fixed bool values
ADDR = code address

fixed values:
    - #NIL
    - #TRUE
    - #FALSE

+================+======================+========================+====================+
|    Instruction |          Opcode args |             Stack Args |       Stack Return |
```
